## ❤️ 기획 배경
<b> 디자이너들의 '감'잡는 커뮤니티 플렛폼 `"GAM"`입니다. </b>  <br> 
디자이너들이 모여, 크리에이티브한 <b> 작업물을 공유하고 소통하는 플렛폼 구축</b> 을 목표로 하고 있습니다. <br>
디자이너 내 <b>`'공유 문화'`</b>를 확산하여, 디자인 커뮤니티의 성장과 발전을 촉진하며, 디자이너들의 위상과 가치를 높이고자 합니다.  <br>
